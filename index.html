Lately, in my spare time outside of work, I have been delving into the world of cryptocurrency.

I'm a big fan of what Loopring is doing with their Layer 2 scaling solution for Ethereum.

One of the main hurdles for a C# developer like myself trying to develop for Loopring, is that there was no C# implementation of the Poseidon hashing function that is used to sign requests to Loopring's API.

As a result of this, I created PoseidonSharp, a Poseidon hashing library for Loopring in C#.

The original reference implementation for Poseidon was in Python so converting this into C# was no easy task to say the least. Let me just say I am not the biggest fan of the BigInteger type in C# as a result...

There is still more work to do to get it into a state I'm happy but initial testing is looking good.
